# RuSpamFilter

Веб‑приложение для определения спама в русскоязычных сообщениях с помощью BERT.

## Описание

Проект представляет собой веб-сервис, который:
- принимает короткий текст на русском языке от пользователя;
- анализирует его с помощью дообученной BERT-модели;
- возвращает бинарную классификацию: «СПАМ» или «НЕ СПАМ».

## Функциональные возможности

- Ввод текста через веб-интерфейс (Gradio).
- Мгновенное определение спама.
- Отображение результата с указанием вероятности принадлежности к классу «спам».

## Ссылка на загрузку модели

Обученная модель доступна на Hugging Face Hub:
[AsuUser/ruSpam_model](https://huggingface.co/AsuUser/ruSpam_model)

При запуске приложения модель загружается автоматически (не требует скачивания).

## Инструкция
1. Проведите усатновку зависимостей: ```pip install -r requirements.txt```
2. Запустить через консоль с помощью команды: ```python src/app.py```
3. Приложение откроется по адресу http://127.0.0.1:7860
